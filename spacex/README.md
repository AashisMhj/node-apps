# SpaceX GraphQl APi
Express Server to Server SpaceX api data on graphql.

## Packages 
* express: Web Server
* axios: Fetching data from spacex api
* cors
* morgan 
* graphql: 
* express-graphql
## Dev packages
* nodemon

## Installation Guide
**Prerequisite**
* Nodejs

```bash
# copy the content of the .env.example and add you stripe keys

# Install packages
yarn
# run dev server
yarn dev:server
```
